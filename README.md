# 📐 MathHub

A fun, interactive web platform to help kids and students practice mathematics.  
From basic arithmetic to more challenging problems – MathHub makes learning math engaging and accessible.

👉 **Live demo:** [athishsreeram.github.io/mathhub](https://athishsreeram.github.io/mathhub/)

## ✨ Features

- **Multiple math topics** – addition, subtraction, multiplication, division, and more.
- **Interactive quizzes** – instant feedback with score tracking.
- **Adjustable difficulty** – suitable for grades 1–6.
- **Progress tracking** (optional) – see how many problems you’ve solved correctly.
- **Kid‑friendly UI** – bright colors, large buttons, and simple navigation.
- **No login required** – start practicing immediately.
- **Responsive design** – works on desktops, tablets, and smartphones.
- **100% client‑side** – no backend, no data collection.

---

## 🚀 How to Use

1. Open the [live demo](https://athishsreeram.github.io/mathhub/) or run locally (see below).
2. Choose a topic (e.g., Addition, Multiplication).
3. Select a difficulty level (Easy / Medium / Hard).
4. Solve the displayed problem by typing your answer.
5. Get immediate feedback – “Correct!” or “Try again”.
6. Track your score and move to the next question.

## 🧪 Local Development

Run the project on your own computer:

```bash
git clone https://github.com/athishsreeram/mathhub.git
cd mathhub
```

Open `index.html` in any modern browser – no build tools or server required.

### Typical file structure
```
mathhub/
├── index.html       # main page and layout
├── style.css        # styling, colors, responsiveness
├── script.js        # quiz logic, question generation, scoring
└── README.md        # this file
```
